Bspwm

bspwm is a popular tiling window manager. It represents windows as the leaves of a full binary tree. bspwm supports multiple monitors and is configured and controlled through messages. It's an alternate window manager in Archcraft.

Desktop

This is your bspwm desktop. It's just a standalone tiling window manager, So there are no desktop icons, Right click functionality. A statusbar (polybar) on the top, shows your various system information, window modes in bspwm and lets you switch between workspaces, control music, volume, brightness, connect to a network, etc.

Menu and Applets

A launcher lets you run apps, runner lets you execute commands quickly. The networkmanager applet lets you manage network connections. music applet lets you control the music player, powermenu applet lets you manage the session. appsasroot applet lets you run apps as root and window applet shows you all running apps.

Applications

PcmanFM is the default file manager, Geany is the text editor, Firefox is the web browser, Alacritty is the terminal emulator and Xfce Terminal as alternate terminal in case alacritty doesn't work for you. There are pre-configured command line applications (vim, ranger, htop, ncmpcpp) installed too.

  • View Keybindings

Hardware Keys

Laptop's function keys are fully configured to use. You can easily control the speaker/headphone volume, Screen brightness. You can control music playback (default to mpd). With print key, You can take screenshots of the desktop (with delays), focused window or selected area.

  • View Keybindings

Window Management

By default, Windows are tiled in bspwm, But windows can also be used in floating mode, full screen mode, etc. Windows can be tiled horizontally and vertically. You can even pre-select the ratio of tiles. Windows can be expanded and shrunk, Marked and locked. You can set the specific rules for a particular application, based on the class name of it's window.

Keybindings

Here's a list of some important keybinds you need to know for operating bspwm easily. If you want to view, modify or add new keybinds, Edit ~/.config/sxhkd/sxhkdrc file.

  • W : Window
  • C : Control
  • S : Shift
  • A : Alt

Applications

Keybind Action

W + S + Return Open floating terminal
W + t Open fullscreen terminal

W + S + w Open web browser (firefox)
C + A + v Open vim (CLI text editor)
C + A + n Open neovim (CLI text editor)
C + A + r Open ranger (CLI file manager)
C + A + h Open htop (system monitor)
C + A + b Open bashtop (system monitor)

Window Management

Keybind Action
W + 1..8 Switch to respective workspace
W + S + 1..8 Move focused window to respective workspace
W + Left/Right/Up/Down Move focused window to respective edge of the screen

W + C + 1..9 Preselect the ratio of tile
W + f Toggle fullscreen mode
W + Space Toggle between floating and tiled mode
W + S + Space Toggle between pseudo tile and tiled mode
W + C + m/x/y/z Set node flag, marked/locked/sticky/private
W + c Close application
W + S + c Kill window
C + A + Escape Kill application
W + Escape Reload Keybinding config
C + A + l Lock the screen
C + S + r Restart bspwm
C + S + q Quit bspwm
