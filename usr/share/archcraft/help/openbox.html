Openbox

Openbox is a lightweight, powerful, and highly configurable stacking window manager. It allows you to change almost every aspect of how you interact with your desktop and invent completely new ways to use and control it. It's the default window manager in Archcraft.

Desktop

This is your desktop. It's just a standalone window manager, So there are no desktop icons. A statusbar (polybar) on the top, shows your various system information and lets you run the launcher, switch to workspaces, control music, volume, brightness, connect to a network, etc. The dock (plank) on the bottom lets you launch your favorite apps quickly and can manage windows. Right clicking on the desktop presents you with the openbox menu, Which allows you to do basically everything (from changing styles to modifying settings).

Menu and Applets

Rofi is used here, to provide all these launchers, menus and applets. A launcher lets you run apps, runner lets you execute commands quickly. The networkmanager applet lets you manage wired/wireless, mobile and bluetooth connections. music applet lets you control the music player, powermenu applet lets you manage the session. There are applets for screenshot and internet too. You can run the launcher and powermenu by clicking the respective module on the polybar.

Applications

Archcraft use lightweight applications, which are fast and do all the work. Thunar is the default file manager, Geany is the text editor, Firefox is the web browser, Alacritty is the terminal emulator and Xfce Terminal as alternate terminal in case alacritty doesn't work for you. There are pre-configured command line applications installed too. vim and neovim text editors, ranger file manager, htop and bashtop system monitors. ncmpcpp for mpd with album-art support, etc.

Hardware Keys

Laptop's function keys are fully configured to use. You can easily control the speaker/headphone volume, Screen brightness. You can control music playback (default to mpd). With print key, You can take screenshots of the desktop (with delays), and by combining it with CTRL, ALT and SHIFT keys, You can capture the active window or selected area. To change the behaviour of these function keys (for ex. music control), Edit ~/.config/openbox/rc.xml file.

  • View Keybindings

Window Management

In openbox, windows float on the desktop, as it's not a tiling window manager. Windows have title bars, control buttons, icons and labels. You can use obconf application to modify the settings. You can switch between different workspaces, move windows to specific workspaces, place windows at different screen edges, You can even tile windows. Easily switch between running apps, move-resize with keyboard shortcuts. You can create specific rules for any application. For more information, view the openbox config file.

Keybindings

Here's a list of some important keybinds you need to know for operating openbox easily. To see all the keybinds, Right click on the desktop and go to Keybinds. If you want to modify or add new keybinds, Edit ~/.config/openbox/rc.xml file.

  • W : Window
  • C : Control
  • S : Shift
  • A : Alt
